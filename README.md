# Four-Bit R-2R Ladder DAC using Multisim

## Project Overview

This project demonstrates the design and simulation of a **4-Bit R-2R Ladder Digital-to-Analog Converter (DAC)** using **NI Multisim**. The R-2R ladder network converts a 4-bit digital input into a corresponding analog output voltage using only two resistor values: **R** and **2R**.

The project provides a practical understanding of Digital-to-Analog Conversion, resistor ladder networks, and their applications in digital electronics and embedded systems.

---

## Objectives

* Design a 4-bit R-2R Ladder DAC.
* Simulate the circuit using NI Multisim.
* Observe the analog output for different digital input combinations.
* Verify the relationship between digital input and analog output voltage.

## Working Principle

The R-2R Ladder DAC consists of a resistor network made using only two resistor values:

* **R**
* **2R**

Each digital input bit controls a switch connected to either the reference voltage (logic HIGH) or ground (logic LOW). The resistor ladder generates a weighted analog output voltage corresponding to the applied binary input.

For a 4-bit DAC:

* Input Range: **0000 to 1111**
* Total Output Levels: **16**
* Analog output voltage increases proportionally with the binary input value.
